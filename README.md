🍲 FoodBridge
----------------------------------------------------------------------------------
* FoodBridge is a web-based platform that connects restaurants, food donors, and NGOs to reduce food waste and help people in need.
* The system allows donors to share surplus food while NGOs can view , accept, or reject donations in real time.

---------------------------------------------------------------------------------------
# 🌟 Project Overview

Food waste and hunger exist side by side in many communities.
FoodBridge provides a simple digital solution where:

- Donors can share surplus food.
- NGOs can find nearby food donations.
- Donations can be accepted in real-time.
- Food images help NGOs verify food quality.
- Donation status is tracked transparently.

# 🍽️ Donation Management

Each donation contains:
- Donor Name
- Food Name
- Food Category
- Quantity
- Location
- Phone Number
- Food Image
- Date & Time
- Status

Donation statuses:
- Available
- Accepted
- Rejected
# 🖼️ Food Image Upload

Donors can upload food images before submitting donations.
Features:

- Base64 Image Storage
- Image Preview in NGO Dashboard
- Image Size Validation
- Supports JPG, PNG and other image formats
